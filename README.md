# Agentic Portfolio

A "Stealth" aesthetic portfolio built with Next.js, featuring terminal interactions and agentic workflow visualizations.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Features

- **Terminal Hero**: Typing animation with system initialization sequence.
- **Operational History**: Project grid with scanning effects and dependency tree popups.
- **Activity Log**: Scrolling system log replacing traditional resume.
- **System Footer**: Industrial-style connection links with hover animations.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
